# jigsaw_gis
mesh generation with jigsaw for the greenland ice sheet

## run

Using a python environment with jigsaw-python installed:

```
python gis.py
```

## dependencies

- jigsaw-python: 1.0.0 # installed from source
- Python: 3.10.10
